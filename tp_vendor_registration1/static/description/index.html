Vendor Registration

Complete vendor registration system with website integration for Odoo 18. Allow potential suppliers to register through your website and manage their requests efficiently in the backend.

Website Integration

Public registration form with validation and thank you page

Backend Management

Complete CRUD operations with Kanban, Tree, and Form views

Approval Workflow

Review, approve, or reject vendor requests with automatic partner creation

Communication Tracking

Built-in chatter for tracking all communications and status changes

Multi-Department Support

Categorize requests by department with additional notes

Purchase Integration

Smart buttons showing related vendor requests in Purchase Orders

Key Features:

  • ✅ Complete website registration form with validation
  • ✅ Automatic partner creation upon approval
  • ✅ Status tracking (Draft → Submitted → Approved/Rejected)
  • ✅ Smart buttons and related records
  • ✅ Security groups for different user access levels
  • ✅ Multi-department categorization
  • ✅ Address and contact information management
  • ✅ Integration with Purchase module
  • ✅ Responsive design for mobile devices
  • ✅ Email notifications and activity tracking
